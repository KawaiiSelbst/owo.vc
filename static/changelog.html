Changelog

2023-04-01

owo.vc's backend has been rewritten from scratch, making the API more RESTful as well as cleaning up the codebase considerably. Old API endpoints are still available for backwards compatibility, but are scheduled for removal on August 1, 2023. This date may be extended if a significant portion of users continue to use these deprecated endpoints.

Notable changes in this release are as follows:

  • API operations have been moved to routes under /api/v2/.
  • Duplicate links shortened using the same method and metadata handling in a short period of time will have the same shortened link returned to them. Previously, a new link was generated even if requests were duplicated.
  • owoify and proxyMetadata have been removed as properties when shortening a link, and have been replaced with the metadata key. See the API documentation for more information.
  • Owoified metadata is now cached for a period of time. If the metadata on the destination page is updated for a link with owoification enabled, changes will not be immediately reflected in the owoified metadata.
